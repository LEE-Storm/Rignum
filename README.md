# Market Chatter Index

Automated index of publicly available market discussions and references.

## What this project is
- A technical indexing and navigation platform
- Catalogs publicly available third-party content
- Displays non-evaluative metadata and external links only

## What this project is NOT
- Not an investment, trading, or financial service
- Not financial advice or analysis
- Not a content publisher or host

## Core principles
- No text reproduction
- No summaries or interpretations
- No sentiment or predictions
- External sources only
- Unverified content

## Status
Early-stage MVP.
